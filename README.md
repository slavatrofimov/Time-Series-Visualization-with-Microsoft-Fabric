## Overview

This solution accelerator helps you analyze time series data in Power BI using an interactive brush-based selection experience combined with a KQL Database in Microsoft Fabric (or Azure Data Explorer).

![Time Series Visualization](docs/assets/Time%20Series%20Brushing.gif)

## Quick Start

### Basic Solution
1. Download or clone this repository
2. Open `src/Time Series Viz - Basic/Time Series Viz - Basic.pbip` in Power BI Desktop
3. Sign in with your Microsoft account
4. Start exploring!

### Advanced Solution
1. Download or clone this repository
2. Open `src/Time Series Viz - Advanced/Time Series Viz - Advanced.pbip` in Power BI Desktop
3. Sign in with your Microsoft account
4. Explore enhanced capabilities!

## Prerequisites

- Power BI Desktop
- Microsoft Account with access to:
  - A KQL Database (Microsoft Fabric or Azure Data Explorer)
  - A Fabric Workspace (to publish and share)

> **Note:** Both solutions come pre-configured with sample data from a publicly-accessible KQL database.
